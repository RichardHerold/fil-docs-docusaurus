The fastest way to get started. No code required.

## 1. Sign in

Go to [console.fil.one](https://console.fil.one) and log in to your account.

## 2. Create a bucket

Click **Create Bucket** on the dashboard. Enter a name (lowercase, no spaces) and click **Create**.

## 3. Upload a file

Open your bucket and click **Upload**. Select a file from your computer. The file appears in the object list once the upload completes.

## 4. Download the file

Click the file name to open the object detail view. Click **Download** to retrieve it.

You have completed your first upload. From here, you can:
- [Generate API keys](/security/authentication) to access your bucket programmatically
- [Enable Object Lock](/storage/object-lock) on a new bucket for immutable storage
- [Explore SDK examples](/sdk/overview) for Python, JavaScript, and Go
