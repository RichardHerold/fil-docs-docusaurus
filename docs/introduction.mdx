Fil Hyperspace is an S3-compatible object storage platform designed for teams that need reliable, cost-effective storage with built-in compliance features. If you've used AWS S3, you already know how to use Fil Hyperspace.

## Why Fil Hyperspace

**S3-compatible from day one.** Use the tools you already know — AWS CLI, boto3, any S3-compatible SDK. Point your existing code at our endpoint and it works.

**Object Lock (WORM compliance) built in.** Protect critical data with immutable retention policies. Compliance mode ensures no one — not even account admins — can delete or overwrite protected objects until retention expires.

**Simple, transparent pricing.** No egress fees, no hidden charges. Pay for what you store.

**30-day free trial.** 1 TB of storage and 2 TB of bandwidth, no credit card required.
